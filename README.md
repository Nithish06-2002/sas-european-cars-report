## European Cars Report (SAS)

### Objective
To generate a formatted report of European cars priced above 30,000 using SAS.

### Dataset Used
- sashelp.cars (SAS built-in dataset)

### SAS Concepts Used
- Comment statements
- TITLE and FOOTNOTE statements
- PROC PRINT
- WHERE clause
- VAR statement

### Tools Used
- SAS Studio (OnDemand for Academics)
- GitHub

### Key Learning
- Filtering rows using WHERE
- Selecting variables using VAR
- Creating clean, readable SAS reports
